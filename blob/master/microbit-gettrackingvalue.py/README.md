## Utiliser comme extension

Ce dépôt peut être ajouté en tant qu'**extension** dans MakeCode.

* ouvrir [https://makecode.microbit.org/](https://makecode.microbit.org/)
* cliquez sur **Nouveau projet**
* cliquez sur **Extensions** dans le menu engrenage
* recherchez **https://github.com/sparco95/microbeat/blob/master/microbit-gettrackingvalue.py** et importez

## Éditer ce projet ![Badge du statut de la compilation](https://github.com/sparco95/microbeat/blob/master/microbit-gettrackingvalue.py/workflows/MakeCode/badge.svg)

Éditer ce dépôt dans MakeCode.

* ouvrir [https://makecode.microbit.org/](https://makecode.microbit.org/)
* cliquez sur **Importer** puis cliquez sur **Importer l'URL **
* collez **https://github.com/sparco95/microbeat/blob/master/microbit-gettrackingvalue.py** et cliquez sur importer
